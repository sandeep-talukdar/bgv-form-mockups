Aadhaar in identity check — This replaces the previous "Include Aadhaar in identity check" toggle from the Add Candidate page. If Aadhaar is checked here, it will appear on the candidate's background verification form. For existing companies, this setting has been migrated from the old toggle value.
Migration note: The old include_aadhaar_in_id_check boolean field on IBGVConfig maps to whether Aadhaar is checked in this Identity document list. Companies with include_aadhaar_in_id_check = 1 → Aadhaar checked. Companies with include_aadhaar_in_id_check = 0 → Aadhaar unchecked. The toggle in the Add Candidate flow should be removed once this config page is live.
