Shared UI Patterns (Engineering Reference)

Sidebar Navigation

Left sidebar (332px) with "Yet to complete" and "Completed" sections. Active item has bg: #f5f7ff + left border #1e40af. Progress bar (140px wide, #eef2ff track, #0f973d fill). Status dots: grey circle = incomplete, green check = complete.

Upload Zone

Dashed border rgba(8,40,204,0.4), 16px radius, 140px height. Contains DigiLocker button (231×44, #6334fa border) + "or" divider + upload icon with "Click to upload" link. Used in Identity, Address, and Education (when doc upload is configured).

Proceed Button

Always INSIDE the sub-section card, right-aligned at the bottom of .subsection-content. 168×44px, #0828cc, 8px radius. Disabled state: opacity: 0.4. Enabled when all required uploads for that sub-section are complete.

Exit Flow Modal

Triggered by "I don't have this document" / "I don't have preferred documents" links. 2-step modal: Step 1 = Reasoning (dropdown + textarea + date). Step 2 = Alternate documents list. See Scenarios 8 and 9.

Mandatory Documents Group

Red-bordered group (#fca5a5) listing all required docs with status pills (Pending / Uploaded). Turns green (#86efac) when all are uploaded. Count badge in header. See Scenarios 3 and 4.

Section Title Card

Top card in content area. 72px min-height, #d9d9d9 border, 12px radius. Title (24px, 600 weight) + inline status dots. Dots match sidebar: grey circle = incomplete sub-section, green check = complete.
